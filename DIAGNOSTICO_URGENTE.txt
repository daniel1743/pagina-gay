🔍 POSIBLES CAUSAS:
═══════════════════════════════════════════════════════════════

CAUSA 1: El orquestador NO se está ejecutando
----------------------------------------------
SÍNTOMA: No aparece "🎭 Conversaciones programadas cada 2 minutos"
SOLUCIÓN: Verificar que schedulePeriodicConversations se esté llamando

CAUSA 2: Los bots están en modo individual (viejo código)
----------------------------------------------------------
SÍNTOMA: Aparecen saludos sueltos como "Hola!", "Buenas!", "¿Qué tal?"
SOLUCIÓN: Verificar que NO se esté llamando startBotActivity

CAUSA 3: triggerBotResponse no se está ejecutando
--------------------------------------------------
SÍNTOMA: Cuando escribes, no aparece ningún log en la consola
SOLUCIÓN: Verificar que ChatPage.jsx esté llamando triggerBotResponse

CAUSA 4: Error de Gemini API
-----------------------------
SÍNTOMA: Error 404 o 400 en la consola
SOLUCIÓN: Verificar API key y modelo

CAUSA 5: Problema con el historial de mensajes
-----------------------------------------------
SÍNTOMA: Los bots responden pero con mensajes vacíos o genéricos
SOLUCIÓN: Verificar que getConversationHistory retorne el array correcto
