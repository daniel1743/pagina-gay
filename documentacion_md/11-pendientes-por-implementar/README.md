# Pendientes Por Implementar

Esta carpeta centraliza dos tipos de documentos:

- pendientes reales detectados que todavia no estan cerrados
- documentos de implementacion y checklists movidos desde otras carpetas para revision operativa

## Objetivo

Evitar que los documentos de trabajo queden dispersos y facilitar una vista unica de:

- que falta implementar
- que ya se documento como implementacion
- que debe revisarse antes de deploy o cierre

## Estructura

- `PENDIENTES_CRITICOS_EDAD_NORMAS_Y_ACCESO_CHACTIVO_2026-04-13.md`
  pendiente critico detectado en acceso guest, verificacion de edad y aceptacion de normas
- subcarpetas tematicas replicadas desde `documentacion_md/*`
  documentos de implementacion movidos para consulta centralizada

## Nota

Esta carpeta no reemplaza los planes maestros ni las auditorias.
Sirve como mesa de trabajo para ejecutar y cerrar implementaciones pendientes.
